\chapter{相关技术与理论基础}

\section{MRI 医学图像与数据特点}
MRI 是一种利用核磁共振现象进行人体组织成像的医学影像技术。与 CT 等成像方式相比，MRI 在软组织成像方面具有较高对比度，并且不涉及电离辐射，因此常用于脑部、脊柱、关节和腹部等部位的结构观察。MRI 数据通常以三维体数据形式存在，一个病例包含多个连续切片，不同切片之间具有空间顺序关系。

医学图像与自然图像相比具有明显差异。首先，医学图像通常灰度分布集中，局部纹理和结构边界具有专业语义，生成结果需要保持解剖结构合理性。其次，医学数据的采集和标注成本较高，公开数据集规模通常小于自然图像数据集。再次，医学数据受设备、扫描协议和人群差异影响，同一模态内部也可能存在分布差异。因此，在医学图像生成任务中，数据预处理和结果解释必须谨慎，不能仅以自然图像生成中的视觉真实性作为唯一评价标准。

\section{NIfTI 数据格式与预处理方法}
NIfTI 是神经影像领域常用的数据格式，支持单文件 \texttt{.nii} 和压缩 \texttt{.nii.gz} 等形式，并能在头文件中保存体素尺寸、空间方向和仿射坐标等信息\cite{nifti2004}。本文使用的 IXI 数据集提供了 NIfTI 格式 MRI 数据，适合进行切片提取和二维生成模型训练\cite{ixiDataset}。

本文的预处理流程包括四个步骤。首先读取 NIfTI 体数据，并沿轴向提取二维切片；其次选取体数据中间部分切片，减少头顶部和边缘低信息区域对训练的影响；然后采用 2\% 到 98\% 百分位归一化抑制极端灰度值，并将像素值映射到 $[0,255]$；最后将切片缩放到统一分辨率并过滤均值、标准差和非零像素比例较低的切片。该流程能够得到灰度范围统一、尺寸一致、信息量相对稳定的 PNG 切片数据。

\section{生成对抗网络基础}
GAN 由生成器 $G$ 和判别器 $D$ 组成。生成器从随机潜变量 $z$ 中生成样本，判别器判断输入样本来自真实数据还是生成器。二者通过极小极大博弈进行优化，其基本目标函数可表示为：

\begin{equation}
\min_G \max_D V(D,G)=\mathbb{E}_{x\sim p_{data}(x)}[\log D(x)] + \mathbb{E}_{z\sim p_z(z)}[\log(1-D(G(z)))] .
\end{equation}

在理想情况下，生成器学习到真实数据分布，判别器难以区分真实样本和生成样本。GAN 的优点是能够生成较清晰的图像样本，但训练过程容易受到模式崩溃、判别器过强和数据量不足等问题影响。医学影像数据集通常规模有限，因此直接训练 GAN 容易出现过拟合或生成样本多样性不足。本文采用 StyleGAN2-ADA，是因为其在高质量图像生成和有限数据训练方面具有较成熟的工程实现。

\section{StyleGAN2 与 ADA 技术}
StyleGAN 系列模型的核心思想是将输入潜变量映射到中间潜空间，再通过风格调制控制生成器不同层的特征表达\cite{karras2019stylegan}。这种结构使模型能够在不同语义层级上控制图像生成过程。StyleGAN2 在 StyleGAN 基础上改进了归一化方式，减少了生成图像中的伪影，并通过路径长度正则化增强潜空间到图像空间映射的稳定性\cite{karras2020stylegan2}。

StyleGAN2-ADA 进一步提出自适应判别器增强机制。当训练数据规模有限时，判别器容易记住训练样本，从而导致生成器无法获得有效梯度。ADA 根据训练过程中的统计信号动态调整数据增强强度，使判别器不易过拟合，同时尽量避免增强操作破坏数据分布\cite{karras2020ada}。本文 MRI 数据集由三维体数据切片得到，虽然切片数量较多，但病例来源和结构类型仍有限，因此采用 ADA 有助于提升训练稳定性。

\section{3D Gaussian Splatting 技术}
3D Gaussian Splatting 是一种三维场景表示和实时渲染方法。其基本思想是使用一组三维高斯基元表示空间中的结构信息，每个高斯包含位置、尺度、方向、不透明度和颜色等参数。通过优化这些高斯参数，可以在保持较高渲染效率的同时表达复杂空间结构\cite{kerbl20233dgs}。

在本文项目中，3DGS 的作用主要体现在三维重建方向的设计上。由于 MRI 数据本身是连续切片组成的体数据，而 StyleGAN2 生成的是二维切片，二者之间需要一个空间组织过程。本文当前实现将连续生成切片按照 $z$ 轴顺序堆叠，并把非背景像素转换为空间点和灰度颜色，生成 PLY 格式初始点云。该点云可作为后续 3DGS 优化或三维可视化的基础输入。需要说明的是，当前阶段的点云初始化并不等同于完整 3DGS 训练结果，真实 3DGS 优化仍属于后续重点工作。

\section{Gradio 交互平台与工程支撑技术}
Gradio 是面向机器学习模型演示和交互测试的 Python 工具，能够快速将模型输入、推理过程和输出结果封装为 Web 界面\cite{abid2019gradio}。对于医学图像生成平台而言，Gradio 的优势在于可以降低模型使用门槛，使用户通过浏览器完成数据路径填写、参数设置、图像生成和结果预览。

本文平台采用 Python 作为主要开发语言，使用 PyTorch 支撑 StyleGAN2 模型训练与推理，使用 PIL 和 NumPy 处理图像和数组，使用 Gradio 构建交互界面，使用子进程方式调度预处理、训练和点云初始化脚本。该技术组合既能复用成熟开源模型框架，也便于将算法流程整合到统一平台中。
