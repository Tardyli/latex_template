\begin{sloppypar}
Medical image generation experiments often face two practical issues. Real MRI data is costly to collect, and a single two-dimensional slice cannot show spatial structure clearly. This thesis addresses these issues by designing and implementing an MRI medical image generation platform based on StyleGAN2 and 3D Gaussian Splatting. The system connects NIfTI preprocessing, MRI slice generation, continuous sequence construction, point cloud initialization, 3DGS training, and interactive visualization in one workflow. The IXI brain MRI dataset is used as the experimental source. The system performs volume slicing, intensity normalization, resolution standardization, low-information slice filtering, and StyleGAN2-ADA dataset construction. It then trains a two-dimensional MRI slice generator and connects the trained model to the platform. For the three-dimensional stage, the system prepares 180 continuous MRI slices and a 300000-point initial point cloud. It completes 30000 iterations of 3DGS training and produces a representation with 572073 Gaussian points. The platform is implemented with a FastAPI and React architecture. The backend provides data, generation, analysis, task, and 3D asset APIs, while the frontend provides the dashboard, generation workbench, medical image viewer, report page, trust evaluation, point cloud display, and AI assistant. Experimental records show that the trained StyleGAN2 model can generate MRI-like slices with recognizable brain grayscale structures. At iteration 30000, the 3DGS training reaches an L1 loss of 0.0709 and a PSNR of 17.5894 dB. The mean global SSIM over 180 rendered-reference slice pairs is 0.8935. These results are based on project logs, model snapshots, and rendered outputs. They show that the platform has built a reproducible engineering pipeline from MRI data processing to two-dimensional generation and three-dimensional Gaussian representation.
\end{sloppypar}
