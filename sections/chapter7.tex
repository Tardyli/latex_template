\chapter{结论与展望}

\section{结论}
本文围绕“基于 StyleGAN2 与 3DGS 的 MRI 医学图像生成平台的设计与实现”开展研究。研究工作覆盖 MRI 原始数据处理、二维切片生成、连续切片序列组织和三维点云构建，并将相关算法脚本整合到 FastAPI + React 前后端分离平台中。

在数据处理方面，系统实现了面向 NIfTI 格式 MRI 数据的预处理流程，能够完成轴向切片提取、百分位归一化、尺寸缩放、低质量切片过滤和训练数据打包。在二维生成方面，系统基于 StyleGAN2-ADA 完成 MRI 切片生成模型训练，并将训练后的权重接入平台推理流程。平台可以根据随机种子和截断参数生成切片。在三维重建方面，系统构建连续 MRI 切片序列、PLY 初始点云和规则相机参数，完成 30000 轮 3DGS 训练，并得到包含 572073 个高斯点的三维表达结果。在平台实现方面，系统使用 FastAPI 构建后端接口，使用 React 构建前端页面，集成结果概览、切片生成、医学影像查看、质量与风险分析、可信性评估、任务中心、三维展示和 AI 助手等功能。各模块共同形成从数据准备到结果展示的完整链路。

实验与测试结果表明，系统具备较完整的教学演示和实验复现能力。StyleGAN2 训练链路已经形成可加载权重，平台能够基于该权重生成 MRI 切片。点云初始化模块能够输出可查看的 PLY 文件。3DGS 模块能够在 180 张连续切片监督下完成训练。第 30000 轮训练达到 L1 为 0.0709、PSNR 为 17.5894 dB，180 对渲染切片与参考切片的平均全局 SSIM 为 0.8935。该结果说明三维高斯表达能够保持 MRI 切片的主体结构，也说明平台具备从二维生成结果进入三维展示的工程基础。

\section{展望}
后续工作可围绕 3DGS 表达能力、评价指标、部署方式和应用场景展开。当前系统已经完成基于 MRI 切片序列的 3DGS 训练和结果导出。下一步可结合医学切片的规则层间关系，优化相机模型、空间约束、正则化策略和高斯点裁剪方式，以提升细节边界和层间连续性。

生成质量评价体系也需要继续扩展。本文已经使用 L1、PSNR、全局 SSIM 和可视化对比分析 3DGS 重建结果。后面可加入 FID、医学专家主观评价、分割一致性或下游任务评价等指标，从视觉质量、结构合理性和任务有效性等角度分析生成图像。

平台部署方式还可以进一步规范。当前系统通过本地 FastAPI 服务和 Vite 前端运行，仍依赖本地数据路径、模型权重路径、GPU 环境和部分外部接口。后面可通过配置文件、环境变量和容器化方式管理运行环境，使平台更适合课程教学、实验复现和多人协作。

应用场景也值得继续拓展。本文主要围绕脑部 T1 MRI 切片生成展开。后面可尝试多序列 MRI、其他公开医学数据集或跨模态图像生成任务，也可以探索生成结果在医学教学、数据增强和三维可视化中的更多应用价值。
