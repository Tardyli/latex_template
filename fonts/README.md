# Thesis font files

This directory is reserved for strict school-format compilation on Overleaf.

Upload the following Windows font files here when compiling on Overleaf:

- `simsun.ttc` for Songti / SimSun
- `simhei.ttf` for Heiti / SimHei
- `simfang.ttf` for Fangsong / FangSong

Do not commit proprietary font files to a public repository unless you have the
right to redistribute them. If the GitHub repository is public, prefer uploading
these font files directly in Overleaf, or make the repository private before
syncing font files through Git.
