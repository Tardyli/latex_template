# Overleaf LaTeX 论文骨架

这是一套根据学校 `docx` 模板结构手工重建的 LaTeX 论文骨架，适合作为：

- Overleaf 新项目的初始目录
- GitHub 论文仓库的起点
- 本地 VSCode + LaTeX 的写作工程

## 当前内容

- `main.tex`：论文主入口
- `sections/`：封面、声明、摘要、章节、附录、致谢
- `bib/references.bib`：参考文献数据库
- `figures/`：图片目录

## 使用建议

1. 先把整个目录上传到 Overleaf。
2. 确认能正常编译。
3. 再按学校模板要求细调字体、字号、间距、页眉页脚。
4. 之后开始逐章写正文，不建议先追求排版像素级一致。

## 编译建议

- 编译器：`XeLaTeX`
- 参考文献：`biber`

## 后续你需要替换的内容

- 中文摘要、英文摘要
